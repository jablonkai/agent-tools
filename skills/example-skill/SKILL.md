# Example Skill

## Purpose

This starter skill demonstrates the expected structure for a reusable skill.

## Use When

- you want to scaffold a new skill quickly
- you need a reference for formatting and section layout

## Inputs

- task goal
- constraints
- expected output format

## Workflow

1. Clarify the goal.
2. Identify reusable instructions.
3. Produce a focused output.
4. Note assumptions and follow-up work if needed.

## Output Guidelines

- prefer concise, executable guidance
- avoid generic filler
- document assumptions explicitly
